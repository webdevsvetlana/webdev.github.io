Информация из Википедии

Формула Видмарка. Шведский химик Эрик Видмарк разработал следующую формулу для определения максимальных теоретически возможных концентраций этанола в крови:

с = A : (m × r)

где:

  • c — концентрация алкоголя в крови в ‰,
  • A — масса выпитого напитка (в пересчете на чистый этанол) в граммах
  • m — масса тела в килограммах,
  • r — коэффициент распределения Видмарка (0,70 — для мужчин, 0,60 — для женщин).



    • Попробуем рассчитать точку Балмера. Это точка показывает теоретическое значение оптимального количество алкоголя, используя исходные данные.
